# Desktop To-Do List Application

## 📌 Description
This is a Desktop-based To-Do List application developed in Java.
It helps users manage their daily tasks by adding, viewing, and deleting tasks in a simple graphical interface.

## 🚀 Features
* Add new tasks
* View task list
* Delete completed tasks
* Simple and user-friendly interface
* 
## 🛠 Technologies Used
* Java
* Swing / AWT (for GUI)
* OOP concepts

## ▶️ How to Run
1. Clone this repository
2. Open the project in IntelliJ IDEA / Eclipse
3. Compile and run the `Main.java` file
